# RAG Complaint Chatbot

An intelligent Retrieval-Augmented Generation (RAG) chatbot for analyzing customer complaints in financial services.

## Features

- Complaint preprocessing
- Semantic search
- ChromaDB vector database
- LangChain retrieval
- HuggingFace embeddings
- Gradio interface
